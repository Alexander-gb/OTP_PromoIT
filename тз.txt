Защита OTP-кодом	

Разработать backend приложение по защите операций кодом.
 Сервис должен предоставлять функции (API) создания операции на защиту, генерации защитного кода на операцию, отправки кода по данной операции адресату
(email / sms / telegram), проверки введенного кода при проведении указанной операции. Время жизни защитного кода должно быть ограничено.
Дополнительно (задачи со звездочкой):
- отправка кода по sms, email или telegram (любая реализация на выбор)
- реализовать настройку времени жизни кода и количества цифр в коде для пользователя с правами доступа администратора
- идеи можно взять с TOTP
